<h1>OS Imaging – Linux Deployment on Legacy Hardware</h1>


<h2>Description</h2>
This project demonstrates full operating system imaging and deployment by replacing Windows 7 with Linux Mint 22.3 on a legacy HP laptop. The workflow included downloading the official ISO, creating bootable installation media using Balena Etcher, configuring BIOS settings to boot from USB, and completing a full operating system installation. This lab showcases foundational IT technician skills, including OS deployment, firmware configuration, boot management, and disk reimaging. 

<br />

<h2>Environments Used </h2>

- <b>Linux Mint 22.3 (Cinnamon Edition)</b>
- <b>Balena Etcher</b>
- <b>Bootable USB Media</b>
- <b>BIOS/UEFI Configuration</b>

<h2>Skills Demonstrated</h2>

- Operating system deployment
- Bootable media creation
- BIOS configuration
- Disk wiping and reimaging
- Hardware troubleshooting
- Linux installation and setup
